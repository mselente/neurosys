# Business Areas copy

Denne mappen inneholder konkret side-copy for de seks første forretningsområdene.

Hver fil er skrevet som et faktisk sideutkast, ikke som en strategibeskrivelse. Derfor inneholder de:

- hero-copy med heading, ingress og CTA-er
- forslag til hvor bilde eller illustrasjon bør brukes
- ferdige tekstseksjoner som kan publiseres med mindre justeringer
- verdi-bullets der dette hjelper skannbarheten
- forslag til støtteblokker lenger ned på siden

Følgende støtteblokker går igjen fordi de bygger troverdighet uten å gjøre siden selvskrytende:

- relevante artikler
- relevante referanser / case
- hva NeuroSYS hjelper med
- CTA til workshop, kontakt eller Agent Platform

## Status

De faktiske sektor-sidene lever nå som HTML i `public/forretningsomrader/` —
ikke som markdown-utkast her. Brief-filene under er beholdt som historisk
kontekst, men er ikke autoritative for hva som ligger på sidene.

Se [`docs/sector-page-playbook.md`](../sector-page-playbook.md) for
gjeldende skriveregler og IA-historikk.
